## Module-1

### Task-1: Basic Mathematical Operations
This task demonstrates basic mathematical operations in Python.
- **Input:** Takes two numbers from the user
- **Operations:** Addition, Subtraction, Multiplication, Division
- **Output:** Displays the result of each operation

### Task-2: Personalized Greeting
This task creates a personalized greeting message.
- **Input:** Takes user's first name and last name
- **Process:** Combines the names into a full name
- **Output:** Displays a welcome message with the user's full name

## How to Run

To run any task, use Python from the command line:

```bash
python Module-1/Task-1.py
python Module-1/Task-2.py
```
